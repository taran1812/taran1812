## 🚀 Featured Projects

### 🔍 [Docifer — Multimodal Document Intelligence](https://github.com/taran1812/projectDocifer)
> Production RAG pipeline for PDF Q&A with grounded citations

- Hybrid dense/BM25 retrieval + cross-encoder reranking over **Qdrant** vector store
- **83.8%** answer correctness · **91.2%** citation grounding · **88.2%** retrieval recall@8
- Stack: `FastAPI` `React 19` `PostgreSQL` `Qdrant` `OpenAI` `Docling` `GitHub Actions`

---

### 🛡️ [AgentGuard — LLM Agent Safety Guardrails](https://github.com/taran1812/projectAgentGuard)
> Policy-aware Transformer system controlling AI agent tool execution at runtime

- Fine-tuned DistilBERT classifiers: **83% decision accuracy** · **89.9% micro F1** risk detection
- Detects: prompt injection · sensitive data exposure · unauthorized tool use · policy violations
- Live demo + published models on 🤗 HuggingFace
- Stack: `Python` `Transformers` `FastAPI` `Gradio` `pytest`
